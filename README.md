# Description
フレームワーク使わずにformの入力処理を書いてみた。

# requirement
- docker
- docker-compose up

# usage
1. git cloneする
2. cloneしたディレクトリの直下で`docker-compose up -d`を実行する
3. localhost:4000にブラウザでアクセスする
